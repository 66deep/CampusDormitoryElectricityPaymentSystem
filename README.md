## 功能介绍（完整见运行截图）
管理员：登录、注册、退出；导航页面管理，包含首页、个人中心、学生管理、宿舍信息管理；电费管理功能，包括用电量查询、电费扣除、充值管理；系统管理功能，例如断电通知管理、公告编辑等。

学生：账号注册和登录；访问主页和个人中心查看或修改个人信息；宿舍电费查询，缴费记录查看，以及线上支付功能；宿舍信息管理，包括宿舍配置、入住人数调整和宿舍图片上传。
